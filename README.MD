# HOW TO USE 

## import code base

```
git submodule update --init --recursive
```
update the code
```
git pull --recurse-submodules
```
## start the servers 

start & rebuild

```
docker-compose up --build
```

start 
```
docker-compose up
```

build 

```
docker-compose build
```
